 - Transaksi Penjualan Produk Barang
	Pilih barang  
	Beli barang (jumlah barang)
	summary
	metode pembayaran
	user bayar (24jam) (0 = user belum bayar)
	upload bukti pembayaran
	konfirmasi admin pembayaran (1= user sudah bayar + upload resi)
	Sistem ngasih tau penjual ada transaksi barang
	penjual upload resi
	admin validasi resi (2 = Pemandu sudah kirim barang + upload resi)
	user konfirmasi penerimaan barang (3 = User sudah menerima barang)
	admin transfer ke pemandu (4 = Admin sudah transfer uang ke pemandu + transaksi selesai)

	konfirmasi admin pembayaran (status 1= user sudah bayar + upload resi)

	admin transfer ke pemandu (status 3 = Admin sudah transfer uang ke pemandu + transaksi selesai)
